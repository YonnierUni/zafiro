# BUSINESS_CONTEXT.md

## Project Name
Zafiro Bar Lounge Website

## Business Name
Zafiro Bar Lounge

## Project Goal
Build a modern website for Zafiro Bar Lounge that works as:
- business presentation
- digital menu
- customer attraction channel
- foundation for future management features

At this stage, focus only on the visual frontend landing page.

## Current Scope
Frontend only.

Do not build the backend yet.

The current goal is to create the landing page and visual structure of the website.

## Architecture
Use React with an MVC-inspired structure.

Suggested folder organization:
- models
- views
- controllers
- components
- assets
- routes
- styles

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- React Router
- Framer Motion

## Design Style
The visual style should feel:
- elegant
- modern
- premium
- nightlife-inspired
- clean and attractive
- mobile-friendly

## Visual Identity
Main concept:
Dark premium lounge style.

Use a combination of:
- black / charcoal backgrounds
- sapphire blue accents
- soft gold or amber highlights
- warm white text
- subtle glow effects
- refined spacing
- smooth animations

## Brand Personality
Zafiro Bar Lounge should feel:
- stylish
- exclusive
- visually attractive
- modern
- welcoming
- premium but not overly formal

## Target Audience
Main audience:
- young adults
- couples
- groups of friends
- people looking for cocktails, music, nightlife, sports nights, and a nice environment

## Business Location
Florencia, Caquetá, Colombia

## Website Purpose
The website should help:
- present the business professionally
- show the menu clearly
- display the atmosphere of the place
- encourage reservations or contact through WhatsApp
- prepare the project for future admin features

## Landing Page Sections
The landing page should include:

1. Hero Section
- business name
- strong headline
- short description
- primary CTA button
- secondary CTA button

2. About Section
- short story or concept of the place
- highlight the atmosphere, cocktails, and experience

3. Featured Menu Section
- showcase selected cocktails
- elegant cards or modern grid layout
- clear names and prices

4. Gallery Section
- visual area for local photos, drinks, ambiance, or placeholders

5. Events / Experience Section
- highlight music, match nights, special events, celebrations, birthdays, or promotions

6. Reservation / Contact CTA
- section inviting users to reserve or contact by WhatsApp

7. Location / Hours Section
- city
- business hours
- contact methods
- future space for map integration

8. Footer
- social media
- contact
- brand name
- rights / closing phrase

## UX Guidelines
The website must:
- look good on mobile first
- feel visually premium
- be easy to scan
- use clear section separation
- avoid clutter
- have smooth scrolling
- use attractive calls to action
- keep text concise and elegant

## UI Guidelines
Prefer:
- rounded cards
- soft shadows
- subtle borders
- large headings
- elegant spacing
- premium buttons
- modern responsive layout
- sections with visual rhythm
- dark theme with contrast

## Do Not
- do not create backend features yet
- do not overcomplicate the design
- do not make it look generic
- do not use a corporate or boring style
- do not overload the page with too much text

## Expected Output
Build the frontend landing page only.

The result should include:
- a modern homepage
- clean reusable React components
- MVC-inspired project organization
- placeholder content ready to be replaced with real business data
- high-quality visual presentation
